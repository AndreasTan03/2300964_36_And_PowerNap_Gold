API Documentation

Below is the detailed documentation for our API:

Get List of Rooms

URL: `/rooms`

Method: GET

Description: Retrieve a list of available rooms.

Response: Status 200 OK

Example:

[
    {
        "room_id": 1,
        "room_type": "Senderan",
        "room_price": 8000
    },
    {
        "room_id": 2,
        "room_type": "Rebahan",
        "room_price": 17000
    },
    {
        "room_id": 3,
        "room_type": "Pules Poll",
        "room_price": 45000
    }
]

Create a Reservation

URL: `/reservation`

Method: POST

Description: Create a new room reservation.

Request Body:

{
    "first_name": "Andreas",
    "last_name": "Tan",
    "phone_number": "123456789",
    "room_type": "Senderan",
    "number_of_rooms": 2,
    "booking_for": "2023-09-15"
}

Response: Status 201 Created

Example Response:

{
    "status": 201,
    "message": "Reservation successfully made",
    "reservation_number": 4,
    "first_name": "Andreas",
    "last_name": "Tan",
    "room_type": "Senderan",
    "booking_for": "2023-09-15",
    "order_price": 16000
}

Update a Reservation

URL: `/reservation/:reservation_number`

Method: PUT

Description: Update an existing room reservation by reservation number. You can also update and delete the reservation by providing the reservation number.

Request Body (Example):

{
    "room_type": "Rebahan",
    "number_of_rooms": 3,
    "date": "2023-09-20"
}

Response: Status 200 OK

Example Response:

{
    "status": 200,
    "message": "Reservation updated successfully"
}

Delete a Reservation

URL: `/reservation/:reservation_number`

Method: DELETE

Description: Delete a room reservation by reservation number. You can also update and delete the reservation by providing the reservation number.

Response: Status 200 OK

Example Response:

{
    "status": 200,
    "message": "Reservation deleted successfully"
}
